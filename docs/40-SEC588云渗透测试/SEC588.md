# 教材简介

本教材专注于云环境中的渗透测试方法和工作流，旨在指导安全从业者完成基于云的攻击和评估。课程内容不仅涵盖云服务提供商（CSP）的限制与架构，还深入探讨云原生（Cloud-Native）、云优先（Cloud First）以及集成了原生云服务的现代应用程序的安全问题。学员将在真实的公有云基础设施上进行动手实验，学习从小型架构到“Web 级别”大规模架构的弹性渗透测试技术，掌握如何通过身份系统、云基础设施和微服务寻找攻击路径 [3-5]。

本课程的内容非常适合以下人群学习：
* 希望获取新技能，或将其现有渗透测试方法论调整并应用于云环境的专业**渗透测试人员**。
* 希望了解公有云或私有云环境中攻防操作思维与方法的 **IT 运维人员、开发人员（DevOps）和系统管理员**。
* 负责防御和监控云端威胁的**安全运营（SecOps）人员**。
* 需要在组织内采购云渗透测试服务、招聘测试人员或管理安全团队，并希望深入了解该流程的**技术管理人员** [4]。
