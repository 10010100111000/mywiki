# 课程简介

SEC560是一门专注于企业级渗透测试与道德黑客技术的实战课程，旨在帮助学员安全、专业地执行模拟真实威胁的综合渗透测试项目。

这门课程的核心学习内容主要包括：

1. **完整的渗透测试生命周期**：从前期的情报侦察和网络扫描开始，逐步掌握获取初始访问权限、后渗透战术、权限提升、横向移动，以及在规避检测的同时保持持久访问的各项技术。
2. **主流安全工具的实战应用**：深入使用 Nmap、Metasploit、Sliver、BloodHound、Impacket 和 Mimikatz 等行业标准工具。
3. **本地与云端混合环境攻击**：不仅涵盖针对本地 Active Directory（活动目录）的深度攻击路径分析与提权，还包括针对微软 Azure/Entra ID 云环境的漏洞利用与身份验证攻击。
4. **综合实战与夺旗赛（CTF）**：通过大量真实场景的漏洞环境进行动手实验，并在最终的 CTF 中综合应用所有技术，展示从初始突破到全面控制域环境的能力
