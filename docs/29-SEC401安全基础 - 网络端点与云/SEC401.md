## 课程简介

SANS SEC401：安全基础 - 网络、端点与云 （**(Security Essentials – Network, Endpoint, and Cloud)**）本教材旨在为学员提供防御系统和网络的关键知识，教授预防攻击和检测对手的最有效步骤。由于组织机构面临着持续的威胁，不仅需要预防攻击，更需要具备及时的检测和响应能力。本教材通过传授信息安全的基础语言和内部运作机制，为您提供可立即应用于工作场所的实用技术，帮助您自信地保护系统、构建基于检测、响应和预防的纵深防御安全计划。

本课程的内容非常适合以下人群学习：
* 希望填补技术信息安全知识空白的安全专业人员。
* 希望超越简单术语和概念、深入理解信息安全的管理人员。
* 主要职责不是安全，但需要了解安全以提高工作效率的运营人员。
* 需要了解如何构建可防御网络以抵御攻击的信息技术（IT）工程师和主管。
* 负责构建和维护可能成为攻击目标的系统的管理员。
* 需要扎实的安全原则基础以最大化工作效率的取证专家、渗透测试人员和审计员。
* 具备一定信息系统和网络背景，且刚接触信息安全领域的新手。

# 1、网络安全与云基础
本部分从网络架构本身出发，深入探讨网络通信的逻辑和物理组件、数据包的底层协议，并全面审视虚拟化、云计算以及人工智能（AI）所带来的新能力与全新攻击面。

## 1.1 SEC401 简介
* 课程大纲、结构与学习逻辑
* 理解网络架构的本质
* 实验：虚拟机设置 (Virtual Machine Setup)

## 1.2 可防御的网络架构
* 概念、逻辑与物理设计
* 网络架构基础 (Network Architecture)
* 针对网络设备的攻击 (Attacks Against Network Devices)
* 网络拓扑与网络设计 (Network Topologies & Network Design)

## 1.3 协议与数据包分析
* OSI 与 TCP/IP 模型对比
* 网络协议概述 (Network Protocols Overview)
* 互联网协议 (IP) 基础与报头分析
* 互联网控制消息协议 (ICMP)
* 传输控制协议 (TCP) 与 用户数据报协议 (UDP)
* 嗅探：网络流量获取与数据包分析
* 实验 1.1：tcpdump

## 1.4 虚拟化、云与人工智能基础
* 虚拟化概述与虚拟化安全 (Virtualization Overview & Security)
* 云计算概述 (IaaS, PaaS, SaaS)
* 云安全与责任共担模型 (Cloud Security & Shared Responsibility)
* 人工智能概述与 AI 安全挑战
* 实验 1.2：Wireshark

## 1.5 保护无线网络
* “无线”通信的普及性及其安全隐患
* 传统无线：IEEE 802.11 标准及其持续演进
* WEP, WPA1, WPA2, WPA3 安全机制与弱点
* 个人局域网 (PAN) 与新兴无线技术
* 实验 1.3：Aircrack-ng

# 2、纵深防御
本部分将建立每个组织都需要的分层防御战略，深入探讨身份认证、访问控制、现代安全框架，并全面解析数据防泄露机制以及移动设备所带来的组织风险。

## 2.1 纵深防御
* 纵深防御概述 (Defense-in-Depth Overview)
* 风险的构成：保密性、完整性与可用性 (CIA)
* 纵深防御策略与核心安全策略
* 云环境中的纵深防御 (Defense-in-Depth in the Cloud)
* 零信任方法论 (Zero Trust Methodology) 与可变信任

## 2.2 身份、访问管理与密码安全
* IAAA：识别、身份验证、授权与问责
* 传统本地与云端单点登录 (SSO)
* 密码管理、技术与存储 (Password Policies & Storage)
* 密钥派生函数 (Key Derivation Functions)
* 密码评估原理与攻击工具 (Hashcat 与 Mimikatz)
* 多因素身份验证 (MFA)、自适应身份验证与 Passkeys
* 本地与云端的特权访问管理 (PAM)
* 实验 2.1：Hashcat

## 2.3 安全框架
* CIS 控制简介与指导原则 (Introduction to the CIS Controls)
* NIST 网络安全框架 (NIST Cybersecurity Framework)
* MITRE ATT&CK 框架与对手战术映射
* 实验 2.2：Cain and Abel

## 2.4 数据防泄露 (DLP)
* 数据丢失 (Loss) 与 数据泄露 (Leakage) 的核心区别
* 勒索软件机制与预防策略 (Ransomware & Preventative Strategies)
* 本地与云端冗余、数据恢复与合规性 (GDPR 与 CCPA)
* 数据防泄露工具与用户活动监控 (User Activity Monitoring)
* 实验 2.3：AppLocker

## 2.5 移动设备安全
* Android 与 iOS 架构及安全特性对比
* 移动端碎片化问题与安全修复流程
* 移动设备管理 (MDM) 与自带设备 (BYOD) 政策
* 越狱、获取 Root 权限与缓解移动端恶意软件

## 3.1 漏洞评估

* 关键性定义与风险计算 (Criticality and Risks)

## 3.2 渗透测试

* 实验 3.1：Nmap

## 3.3 攻击与恶意软件
* 高瞩目的违规事件案例分析
* 勒索软件即服务 (Ransomware as a Service)
* 常见攻击技术与缓冲区溢出基础
* 恶意软件类别与恶意软件分析 (Malware and Analysis)
* 实验 3.2：恶意软件分析

## 3.4 Web 应用安全
* Web 通信基础：Cookies 与 HTTPS
* 开发安全的 Web 应用与安全编码基础
* OWASP Top 10 与 Web 应用漏洞剖析
* Web 应用监控与 Web 应用防火墙 (WAF)
* 实验 3.3：命令注入 (Command Injection)

## 3.5 安全运营与日志管理
* 日志记录概述、收集架构与过滤
* 解决日志记录标准的常见问题
* 设置和配置日志记录最佳实践
* 日志聚合工具与 SIEM (Log Aggregation and SIEM)
* 关键日志活动审查机制

## 3.6 数字取证与事件响应 (DFIR)
* 数字取证简介与实践操作
* 调查过程与保持取证环境完好性 (Forensically Sound)

* 事件处理基础与多步骤响应流程
* 威胁搜寻 (Threat Hunting) 基础
* 实验 3.4：hping3

# 4、数据安全技术
本部分重点研究防御安全核心的技术手段，从最容易被误解的密码学开始，一直延伸到网络防火墙、入侵检测与端点设备的安全保护。

## 4.1 密码学基础
* 密码系统基础与密码分析 (Cryptanalysis)
* 密码系统类型：对称、非对称与哈希 (Symmetric, Asymmetric, Hashing)
* 数字签名工作原理 (Digital Signatures)
* 实验 4.1：图像隐写术 (Image Steganography)

## 4.2 密码学算法与部署
* 强密码学的数学特征与算法剖析 (AES, RSA, ECC)
* 常见的密码学攻击手法
* 量子计算威胁与后量子密码学 (PQC)

## 4.3 应用密码学
* 传输中的数据保护：VPN、IPsec 与基于 SSL 的加密
* 静态数据保护：文件/文件夹级别加密与全盘加密
* 密钥管理与公钥基础设施 (PKI) 原理
* 数字证书、证书颁发机构与 PKI 常见用例
* 实验 4.2：GNU Privacy Guard (GPG)

## 4.4 网络安全设备
* 防火墙概述、类型与部署配置注意事项
* 网络入侵检测系统 (NIDS) 的类型与应用 (Snort)
* 网络入侵防御系统 (NIPS) 部署方法与生产力风险
* 深度与浅层数据包检测 (Deep vs. Shallow Packet Inspection)
* 实验 4.3：Snort

## 4.5 端点安全
* 端点安全概述与核心防御组件
* 端点安全解决方案：防恶意软件与端点防火墙
* 完整性检查机制 (Integrity Checking)
* 主机入侵检测 (HIDS)、防御 (HIPS) 与 端点检测响应 (EDR)
* 实验 4.4：哈希算法应用 (Hashing)
