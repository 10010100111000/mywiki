# 课程简介

**SANS SEC504：黑客工具、技术与事件处理 (Hacker Tools, Techniques, and Incident Handling)** 是一门旨在全面提升网络安全防御与响应能力的实战型课程。

主要内容分为以下几个核心板块：

**事件响应与网络取证 (Incident Response and Cyber Investigations)**： 引入现代动态事件响应（DAIR）模型，学习如何使用原生工具（如PowerShell）、网络数据包分析工具（Tcpdump）以及内存取证工具（Volatility）对受损系统进行实时调查和脱机分析。同时，探索如何利用大型语言模型（LLM）等生成式AI加速事件响应过程。

**扫描与枚举攻击 (Scanning and Enumeration Attacks)**： 站在攻击者视角，探讨前期攻击策略、技术和程序（TTPs）。包括使用 Nmap 和 Masscan 进行本地与云端资产的网络发现，针对 SMB 等核心协议的枚举漏洞，以及防守方如何利用 Hayabusa 结合 Sigma 规则进行高效的 Windows 威胁分析。

**密码攻击与漏洞利用框架 (Password Attacks and Exploit Frameworks)**： 剖析攻击者最常使用的身份验证攻击，包括密码喷洒、凭据填充及密码哈希破解（使用 Hashcat 和 GPU 加速）。此外，深入学习 Metasploit 漏洞利用框架的模块化攻击与 Meterpreter 后渗透技术，以及利用 BeEF 框架实施的客户端浏览器水坑攻击。

**Web应用程序攻击 (Web Application Attacks)**： 聚焦面向公众的 Web 服务漏洞利用，涵盖不安全的直接对象引用（IDOR）、命令注入、跨站脚本（XSS）以及自动化 SQL 注入攻击。针对云原生环境，课程还专门探讨了服务器端请求伪造（SSRF）、云实例元数据服务（IMDS）漏洞及配置不当的云存储桶发现。

**逃避与后渗透攻击 (Evasion and Post-Exploitation Attacks)**： 研究攻击者在获取初始访问权限后的高级战术。包括利用合法工具（LOLBAS）绕过端点安全和应用程序白名单、实施内网跳板与横向移动、利用 Responder 劫持局域网凭证，以及建立系统持久性。防守方将学习如何使用 RITA 等工具分析网络信标（Beaconing）以识别隐蔽威胁。

**实战夺旗挑战赛 (Capture the Flag Event)**： 课程通过大量的动手实验（Hands-on Labs）和高频短时的“闪电实验（Lightning Labs）”强化技能转化。在课程的最后，学员将参与一场综合性的 CTF 实战工作坊，在模拟真实网络入侵的场景中扮演事件响应分析师，将所学的所有黑客工具与防御技术融会贯通。

这门课专为需要快速且显著提升事件处理技能的专业人士设计，非常适合以下岗位或有相关职业发展意向的人员：

- 安全运营中心 (SOC) 分析师
- 事件响应人员 (Incident Handlers)
- 希望提升安全视角的系统管理员、运维分析师或网络专业人员
- 准备涉足数字取证 (Forensics) 或进攻性安全操作 (Offensive Operations) 领域的从业者
