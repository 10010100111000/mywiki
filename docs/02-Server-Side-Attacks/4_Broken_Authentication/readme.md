# 模块描述

身份验证可能是用于保护资源访问、最直接也最普遍的安全措施，也是抵御未授权访问的第一道防线。

身份验证失效在 2021 OWASP Top 10 中位列第 7 位，归属于更宽泛的 标识与身份验证失败（Identification and Authentication Failures） 大类。身份验证环节出现的漏洞或配置不当，会直接影响整个应用的安全性。
模块总结
本模块讲解与身份验证相关的常见漏洞和配置缺陷，攻击者可利用这些问题实现对 Web 应用的未授权访问。具体内容包括：
身份验证机制概述及针对身份验证的攻击方式
针对登录接口的暴力破解攻击
基于密码的身份验证攻击
身份验证绕过
会话机制不当导致的相关攻击
