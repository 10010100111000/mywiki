模块说明
GraphQL 是一种面向 API 的查询语言，可作为 REST API 的替代方案。客户端能够通过 GraphQL 查询语句获取所需数据。若该技术的配置或代码实现存在缺陷，可能会引发 ** 信息泄露、SQL 注入、不安全直接对象引用（IDOR）** 等常见 Web 安全漏洞。
模块概述
本模块讲解 GraphQL API 中存在的常见错误配置与安全漏洞。
具体涵盖以下核心内容：
GraphQL API 信息枚举
GraphQL API 中的信息泄露漏洞
GraphQL API 中的 IDOR 漏洞
GraphQL API 中的注入类漏洞
GraphQL API 中的拒绝服务（DoS）与暴力破解漏洞
GraphQL API 中的常见错误配置

顺利完成本模块的学习，需要你熟练掌握以下前置模块内容：
Web 攻击基础
跨站脚本（XSS）
SQL 注入基础
