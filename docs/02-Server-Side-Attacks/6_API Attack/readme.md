# 模块说明

Web API 是现代数字生态中连接各类主体的关键枢纽。但其丰富的功能也使其面临多种潜在攻击。本模块将介绍 API 攻击，重点讲解 **OWASP API Security Top 10 - 2023**。

# 模块概述

Web 应用程序编程接口（API）无处不在，可实现互联网上不同系统与应用间的流畅数据交互。但 API 也可能存在各类安全漏洞。本模块深入讲解 API 安全核心领域，探究常见漏洞与攻击向量。
我们将围绕 **OWASP API Security Top 10 - 2023** 展开，通过攻击某电商平台的 RESTful API，分析当前 API 面临的最常见风险。

本模块将详细讲解以下内容：

- API1:2023 对象级别授权失效
- API2:2023 身份认证失效
- API3:2023 对象属性级别授权失效
- API4:2023 无限制资源消耗
- API5:2023 功能级别授权失效
- API6:2023 敏感业务流程未受限制访问
- API7:2023 服务端请求伪造（SSRF）
- API8:2023 安全配置不当
- API9:2023 接口清单管理不当
- API10:2023 不安全地调用第三方 API

- Web 请求
- Web 应用入门

- SQL 注入基础
