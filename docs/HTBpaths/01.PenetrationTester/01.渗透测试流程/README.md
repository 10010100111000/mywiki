# 模块说明
本模块讲解渗透测试流程，将其拆解为各个阶段并展开详细讨论。我们会覆盖渗透测试人员在测试过程中所需承担的多项工作职责，并通过详实案例进行讲解与演示。模块还包含渗透测试项目前期准备相关内容，例如与客户签订渗透测试服务合同的相关标准与要求。

# 模块总结
本模块详细讲解完整的渗透测试流程，并结合案例突出其中核心环节。由于渗透测试可能对部分系统造成潜在影响，我们必须提前做好自身及客户方的准备工作，包括与客户拟定并签订合同，明确需要关注的要点及需要考量的各个方面。

在本模块中，我们将学习：
- 渗透测试流程
- 测试前期准备
- 信息收集
- 漏洞评估
- 漏洞利用
- 利用后操作
- 横向移动
- 概念验证（PoC）
- 测试后期收尾
- 实操练习
