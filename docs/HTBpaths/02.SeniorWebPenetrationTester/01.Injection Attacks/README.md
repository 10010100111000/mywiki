## 模块描述

本模块涵盖三种注入攻击：XPath 注入、LDAP 注入和 PDF 生成库中的 HTML 注入。XPath 和 LDAP 注入漏洞可能导致身份验证绕过和数据泄露，而 PDF 生成库中的 HTML 注入则可能导致服务器端请求伪造 (SSRF)、本地文件包含 (LFI) 和其他常见的 Web 漏洞。我们将介绍如何识别、利用和防御每种注入攻击。

## 模块概要

存在许多注入漏洞，其中最著名的包括 SQL 注入、跨站脚本攻击和命令注入。虽然这些漏洞在实际 Web 应用中最为常见，但还有一些鲜为人知的注入漏洞。由于这些注入漏洞通常不太常见且鲜为人知，开发人员很可能对其毫不知情，因此也往往忽略了相应的缓解措施。如果我们能找到一个使用了 XPath、LDAP 或 PDF 生成库的网站，对其进行针对这些攻击的测试或许能够帮助我们推进项目进展。

本模块具体涵盖以下内容：

- ==XPath Injection== ：
  - XPath 语法简介
  - 利用 XPath 注入绕过身份验证
  - 利用 XPath 注入窃取数据
  - 利用盲注和基于时间的 XPath 注入来窃取数据
- ==LDAP Injection== ：
  - LDAP 语法简介
  - 利用 LDAP 注入绕过身份验证
  - 利用 LDAP 注入窃取数据
- PDF生成库中的HTML注入:
  - PDF 生成库中 HTML 注入简介
  - 利用 PDF 生成漏洞进行服务器端请求伪造 (SSRF)
  - 利用 PDF 生成漏洞导致本地文件包含 (LFI) 攻击
