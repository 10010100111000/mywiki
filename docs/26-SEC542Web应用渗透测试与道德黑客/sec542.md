# 课程简介

SANS SEC542（Web应用渗透测试与道德黑客 / Web App Penetration Testing and Ethical Hacking）是一门全面且注重实战的安全评估课程。通过详细的实战演练，学员将学习Web应用渗透测试的四步流程，掌握如何将SQL注入后端数据库，并利用跨站脚本（XSS）等攻击手段来突破目标基础设施。

该课程的内容涵盖了现代Web应用渗透测试的核心领域，主要包括以下几个模块：

**信息收集与准备**：介绍拦截代理工具（如ZAP和Burp Suite）、开源情报（OSINT）收集、虚拟主机发现、HTTP/HTTPS协议分析及目标画像。

**内容发现与身份验证测试**：涵盖Web爬虫、强制浏览（目录爆破）、模糊测试（Fuzzing）、信息泄露、用户名收集、身份验证与授权绕过，以及会话管理机制的测试。

**注入攻击**：深入讲解命令注入、本地/远程文件包含（LFI/RFI）、不安全的反序列化，并包含专门的SQL注入全方位教学（发现、利用以及sqlmap自动化工具的使用）。

**客户端与特定协议攻击**：重点研究基于文档对象模型（DOM）的攻击、不同类型的跨站脚本（XSS）、浏览器利用框架（BeEF）、异步请求（AJAX）与数据攻击、服务端请求伪造（SSRF）以及XML外部实体注入（XXE）。

**高级攻击与工具**：探讨跨站请求伪造（CSRF）、业务逻辑漏洞，并学习如何使用Python脚本实现测试自动化。此外，还会讲解WPScan、Burp Scanner、Metasploit、Nuclei等高级工具的运用，以及如何在工具失效时进行手工突破，最后涵盖渗透测试的商业流程和报告编写。
